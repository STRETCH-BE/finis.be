## Deploy in 5 Minutes

### STEP 1: Create GitHub Repository
1. Go to https://github.com/new
2. Repository name: `finis-website`
3. Make it **Public**
4. Click "Create repository"

### STEP 2: Upload Files
1. Click "uploading an existing file"
2. Drag and drop ALL 6 files:
   - index.html
   - solutions.html
   - realizations.html
   - style.css
   - script.js
   - README.md
3. Click "Commit changes"

### STEP 3: Enable GitHub Pages
1. Go to Settings (top menu)
2. Click "Pages" (left sidebar)
3. Under Source: Select "main" branch
4. Click "Save"
5. Wait 2 minutes

### STEP 4: Visit Your Live Site!
Your website will be at:
```
https://YOUR-USERNAME.github.io/finis-website/
```

## ðŸŒ Using Your Own Domain

To use www.finis-acoustic.com instead of GitHub URL:

1. **Add DNS Record at your domain provider:**
   - Type: CNAME
   - Name: www
   - Value: YOUR-USERNAME.github.io

2. **Configure in GitHub:**
   - Settings > Pages > Custom domain
   - Enter: www.finis-acoustic.com
   - Check "Enforce HTTPS"

Wait 24 hours for DNS to update.
